# Orbit

## Topic
- Travel planning app

## Description 
- Orbit is a travel planning application that provides recommendations on how to organize your journey and plans your budget for a seamless travelling experience.

## Team Members

## Features 
- Budget tracking
- Ai Itinerary builder
- Onboarding quiz
